# Simple_kanban_board
A Kanban board is a visual tool used to manage work and track the status of tasks in a project. It uses columns to represent stages of a process and movable cards to represent individual tasks, allowing teams to see exactly what is being done, what needs to be done, and what is finished at a glance.


In the making of this Kanban Board I learnt about some new HTML Attributes Like "Draggable"  which makes the content inside the element draggable and dropable.
